# stickers/

表情贴纸系统，根据宠物状态和游戏结果自动弹出对应的 meme 图。

| 文件夹 | 触发条件 |
|--------|---------|
| `moodHigh/` | exp很开心，心情高兴 |
| `moodLow/` | exp很悲伤，心情低落 |
| `hungerHigh/` | exp吃得饱饱~ |
| `hungerLow/` | exp好饿，会变成沙丘吗？ |
| `thirstHigh/` | exp水满满 |
| `thirstLow/` | exp好渴渴 |
| `rpsGameWin/` | 与exp猜拳获胜，exp:“我还会回来的” |
| `rpsGameDraw/` | 与exp平分秋色 |
| `rpsGameLose/` | 与exp猜拳失败，你猜不过我你信吗？ |

**说明**：
- 支持 `.png`、`.jpg`、`.gif`（动图循环播放）
- 放入对应分类文件夹即可，随机弹出

> 说白了这其实就是作者用来收集 Casualties Unknown 的 meme 图的地方，快用你的 meme 喂我🥜（花生）！！
